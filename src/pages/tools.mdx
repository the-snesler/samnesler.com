Miscellaneous low-stakes TypeScript tools built mostly with LLMs. These are tools that I spent some time building for fun, but aren't necessarily groundbreaking enough to put on my main portfolio page. Nonetheless, I hope you find them useful or interesting!

- [Bike Benefits map](https://bb.samnesler.com) - A map of all the businesses that offer Bike Benefits discounts in the US, optimized for mobile.
- [Minecraft Fractal](https://the-snesler.github.io/minecraft-fractal/) - A fun experiment that presents a grid of Minecraft block textures, then lets you zoom in smoothly, infinitely, to see more and more blocks that replace the pixels of the top-level blocks.
- [Paintball](https://paintball.samnesler.com) - An alternative frontend for various image generation models that lets you bring your own keys.
- [Restaurant receipt splitter](/tools/bill-splitter) - A tool that scans a restaurant receipt and makes it easy to assign different entrees to different people. Finally, it calculates tip + tax and splits the bill accordingly.
- [SimpleGIF](https://simplegif.samnesler.com) - A streamlined tool for creating, converting, cropping, and compressing GIFs.
- [Super Brothers Reader](/tools/sb-reader.html) - A simple reader for the [Super Brothers](https://www.deviantart.com/dogmansp/gallery/62677294/super-brothers) webcomic, since DeviantArt is painful for comic reading.
- [Trakt iCal generator](https://trakt.samnesler.com) - A simple tool that generates an iCal feed of your Trakt calendar, so you can stay on top of movie and show release dates.
- [Vibe Check](https://vibecheck.samnesler.com) - Generates an API key that you can use with an LLM agent (like OpenClaw or Poke) to share your location and Discord status.
